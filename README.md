# bootcamp-programador-de-software-iniciante
 Bootcamp do Instituto de Gestão da Tecnologia de Informação - IGTI

<h1>Módulos</h1>

<h2>01 - Aquecimento e Regras do Jogo</h2>

<ul>
 <li>Visão geral da dinâmica do curso, conteúdos e ferramentas utilizadas.</li>
</ul>

<p><strong>Duração: 4 horas</strong></p>

<h2>02 - Fundamentos</h2>

<ul>
 <li>Introdução a lógica de programação;</li>
 <li>Como um programa de computador funciona;</li>
 <li>Instalação do Node.js, nodemon e VS Code;</li>
 <li>Introdução a algoritmos;</li>
 <li>Programação do JavaScript no navegador;</li>
 <li>JavaScript - tipos, valores, variáveis e comentários;</li>
 <li>JavaScript - estruturas de decisão;</li>
 <li>JavaScript - estruturas de repetição;</li>
 <li>JavaScript - funções;</li>
 <li>JavaScript - leitura/escrita de arquivos;</li>
 <li>JavaScript - introdução ao HTML;</li>
 <li>JavaScript - introdução ao CSS;</li>
 <li>JavaScript para a web;</li>
 <li>Manipulação de HTML, CSS e Eventos com JavaScript.</li>
</ul>

<p><strong>Duração: 32 horas</strong></p>

<h2>03 - Java</h2>

<ul>
 <li>Introdução ao Java;</li>
 <li>Java x JavaScript;</li>
 <li>Aplicações do Java</li>
 <li>Instalação e configuração de ferramentas e ambiente de execução de código Java</li>
 <li>Java - tipos, valores, variáveis e comentários;</li>
 <li>Java - estruturas de decisão;</li>
 <li>Java - estruturas de repetição;</li>
 <li>Java - métodos;</li>
 <li>Java - classes e objetos.</li>
</ul>

<p><strong>Duração: 32 horas</strong></p>

<h2>04 - Algoritmos e Estrutura de Dados com Java</h2>

<ul>
 <li>Introdução as estruturas de dados;</li>
 <li>Java - Pilha;</li>
 <li>Java - Fila;</li>
 <li>Java - ArrayList;</li>
 <li>Java - algoritmos de ordenação;</li>
 <li>Java - algoritmos de pesquisa;</li>
 <li>Java - processamento de arquivos.</li>
</ul>

<p><strong>Duração: 32 horas</strong></p>

<h2>05 - Python</h2>

<ul>
 <li>Introdução ao python;</li>
 <li>Tipos de dados em python;</li>
 <li>Bibliotecas e operações básicas em python;</li>
 <li>Introdução a análise de dados;</li>
 <li>Pandas e numpy para análise de dados;</li>
 <li>Introdução ao scikit-learn.</li>

<h2>06 - Desafio final</h2>
